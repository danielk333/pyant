PyAnt
=====

:Release: |release|
:Date: |today|

PyAnt is a standardization of implementing radar gain pattern in Python. Nothing more, nothing less.


Getting started
-----------------

To install (Not yet available)

.. code-block:: bash

   pip install pyant

or 

.. code-block:: bash

   git clone git@github.com:danielk333/pyant.git
   cd pyant
   pip install .

Then get started with this quick "getting started" guide
