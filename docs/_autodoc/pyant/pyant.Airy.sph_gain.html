Airy.sph_gain¶

Airy.sph_gain

Airy.sph_gain(azimuth, elevation, radians=None)¶

Return the gain in the given direction.

Parameters

  • azimuth (float) – Azimuth east of north to evaluate gain in.

  • elevation (float) – Elevation from horizon to evaluate gain in.

  • radians (bool) – If True all input/output angles are in radians, if False degrees are used. Defaults to instance settings self.radians.

Returns

Radar gain in the given direction.

Return type

float
