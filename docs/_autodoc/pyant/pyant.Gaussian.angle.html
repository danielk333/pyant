Gaussian.angle¶

Gaussian.angle

Gaussian.angle(k, radians=None)¶

Get angle between local direction and pointing direction.

Parameters

  • k (numpy.ndarray) – Direction to evaluate angle to.

  • radians (bool) – If True all input/output angles are in radians, if False degrees are used. Defaults to instance settings self.radians.

Returns

Angle between pointing and given direction.

Return type

float
