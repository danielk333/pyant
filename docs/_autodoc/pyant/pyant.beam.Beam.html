Beam¶

class pyant.beam.Beam(azimuth, elevation, frequency, radians=False, **kwargs)[source]¶

Bases: abc.ABC

Defines the radiation pattern of a radar station.

Parameters

  • frequency (float) – Frequency of radiation pattern.

  • azimuth (float) – Azimuth of pointing direction.

  • elevation (float) – Elevation of pointing direction.

  • radians (bool) – If True all input/output angles are in radians, else they are in degrees

Variables

  • frequency (float) – Frequency of radiation pattern.

  • azimuth (float) – Azimuth of pointing direction.

  • elevation (float) – Elevation of pointing direction.

  • radians (bool) – If True all input/output angles are in radians, else they are in degrees

  • pointing (numpy.ndarray) – Cartesian vector in local coordinates describing pointing direction.
