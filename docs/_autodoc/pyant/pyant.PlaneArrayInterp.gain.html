PlaneArrayInterp.gain¶

PlaneArrayInterp.gain

PlaneArrayInterp.gain(k)¶

Return the gain in the given direction. This method should be vectorized.

Parameters

k (numpy.ndarray) – Direction in local coordinates to evaluate gain in. Must be a (3,) vector or a (3,n) matrix.

Returns

Radar gain in the given direction. If input is a (3,) vector, output is a float. If input is a (3,n) matrix output is a (n,) vector of gains.

Return type

float/numpy.ndarray
