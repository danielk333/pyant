pyant.coordinates.cart_to_sph(vec, radians=False)[source]

Convert from Cartesian coordinates (east, north, up) to Spherical coordinates (azimuth, elevation, range) in a degrees east of north and elevation fashion

Parameters

  • vec (numpy.ndarray) – Vector of Cartesian coordinates (east, north, up). This argument is vectorized in the second array dimension, i.e. it supports matrix (3,n) inputs as well as the standard (3,) vector inputs.

  • radians (bool) – If True all input/output angles are in radians, else they are in degrees

Returns

Vector of Spherical coordinates (azimuth, elevation, range)

pyant.coordinates.vector_angle(a, b, radians=False)[source]

Angle in between two vectors \(\theta = \cos^{-1}\frac{\langle\mathbf{a},\mathbf{b}\rangle}{|\mathbf{a}||\mathbf{b}|}\), where \(\langle\mathbf{a},\mathbf{b}\rangle\) is the dot product and \(|\mathbf{a}|\) denotes the norm.

Parameters

  • a (numpy.ndarray) – Vector \(\mathbf{a}\).

  • b (numpy.ndarray) – Vector \(\mathbf{b}\). This argument is vectorized in the second array dimension, i.e. it supports matrix (3,n) inputs as well as the standard (3,) vector inputs.

  • radians (bool) – If True all input/output angles are in radians, else they are in degrees

Returns

Angle \(\theta\) between vectors \(\mathbf{a}\) and \(\mathbf{a}\).

Return type

float.
