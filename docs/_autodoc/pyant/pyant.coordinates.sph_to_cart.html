sph_to_cart¶

sph_to_cart

pyant.coordinates.sph_to_cart(vec, radians=False)[source]¶

Convert from spherical coordinates (azimuth, elevation, range) to Cartesian (east, north, up) in a degrees east of north and elevation fashion

Parameters

  • vec (numpy.ndarray) – Vector of Cartesian Spherical (azimuth, elevation, range). This argument is vectorized in the second array dimension, i.e. it supports matrix (3,n) inputs as well as the standard (3,) vector inputs.

  • radians (bool) – If True all input/output angles are in radians, else they are in degrees

Returns

Vector of Cartesian coordinates (east, north, up)

Return type

numpy.ndarray
