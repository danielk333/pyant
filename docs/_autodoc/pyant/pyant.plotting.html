pyant.plotting.gain_heatmap(beam, resolution=201, min_elevation=0.0, levels=20, ax=None, vectorized=True)[source]

Creates a heatmap of the beam-patters as a function of azimuth and elevation in terms of wave vector ground projection coordinates.

Parameters

  • beam (Beam) – Beam pattern to plot.

  • resolution (int) – Number of points to divide the wave vector x and y components into, total number of calculation points is the square of this number.

  • min_elevation (float) – Minimum elevation in degrees, elevation range is from this number to \(90^\circ\). This number defines the half the length of the square that the gain is calculated over, i.e. \(\cos(el_{min})\).

  • levels (int) – Number of levels in the contour plot.

  • vectorized (bool) – Use vectorized gain functionality to calculate gain-map.

Returns

matplotlib axis and figure handles
