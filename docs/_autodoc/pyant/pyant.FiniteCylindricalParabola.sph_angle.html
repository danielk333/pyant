FiniteCylindricalParabola.sph_angle¶

FiniteCylindricalParabola.sph_angle

FiniteCylindricalParabola.sph_angle(azimuth, elevation, radians=None)¶

Get angle between azimuth and elevation and pointing direction.

Parameters

  • azimuth (float) – Azimuth east of north to measure from.

  • elevation (float) – Elevation from horizon to measure from.

  • radians (bool) – If True all input/output angles are in radians, if False degrees are used. Defaults to instance settings self.radians.

Returns

Angle between pointing and given direction.

Return type

float
